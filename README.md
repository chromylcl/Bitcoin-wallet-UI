# 💰 Bitcoin Wallet UI

A simple Bitcoin wallet interface that allows users to view balance, transactions, and activity using public blockchain APIs.

---

## 🚀 Features

* 🔍 Fetch wallet balance using address
* 📊 Display recent transactions
* ➕ Show received/sent BTC amounts
* ⏱️ Human-readable transaction time
* 🔗 Clickable transaction links (mempool explorer)
* 📋 Copy transaction details

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript (Vanilla)
* Mempool API
